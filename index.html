v1.2 (2019.02.25)

General:

  • Fixed a bug that "silently" crashed the app. You should feel a significant smoothness in scrolling on a row or between rows, exp.: on the Main screen
  • Moved most of the API loadings to Ion, this speeds up loading/reload times
  • ExoPlayer (2.9.5)
  • Setting: Auto refresh (explained in the Settings, ON by default and 20min)
  • Setting: UI Themes (red, blue, purple, green, pink, orange, yellow, grey)

Chat:

  • Improved Chat startup time
  • Significantly improved Emote loading times. This puts the app on par with the desktop experience. It's able to keep up even with xQc's "quality chat".
  • Setting: Show/hide Gifted subs messages in the chat (ON by default)
  • Improved readability of sub and gifted sub messages
  • FFZ channel enabled emotes are activated
  • Setting: Open soft keyboard when the send Chat message window is opened from the UI (ON by default). If you open the chat window with the keyboard shortcut (J or F) this will not affect you, even when the setting is ON.
  • Setting: Filter out popular bots and they command from chat
  • Setting: Filter 'chat bots' by name (OFF by default)
  • Setting: Join Chat (ON by default)
  • Setting: Chat delay (no delay by default)
  • Updated the ASCII dick filter with new characters (yes, there is such thing)

Notification: (NEW)

  • You get a notification when your followed streamer went live. This works even when the app is in the background.
  • The check runs every minute.

Stream:

    PeekRow: (NEW)

  • With the DPAD_UP button on your remote, you can open a row of cards on the bottom of the screen while watching a stream. The row contains the Main screen's "Followed" row, so you can switch between your Followed online streams without going back to the Main screen.
  • Few settings added under Stream -> PeekRow

    • General:

  • Hosting is now supported, so if your current streamer starts to host someone else, the app automatically redirects to the hosted streamer.
  • Better playback error feedback, this is true for VOD as well.
  • Stat for nerds button: if you turned on the feature, you can open it now from the UI.
  • Settings: set default VOD video quality, Auto or Source (Default is Auto)

Fix:

  • Followed channels and games limit 25->100
  • Follower count formatting
  • If you are subbed to a channel, but you still can't access the sub-only VODs, a logout+login should fix that. In the near future i will fix this.
  • Fixed an issue that caused lower fps(<30fps) in the Stream Category while scrolling on Oreo
  • Fixed an issue when opening a Channel profile, colors didn't loaded in a certain situation
  • Fixed an issue when you couldn't see chat, because you had special characters in your name
  • Fixed a unresponsive UI bug on Oreo. Previously, after a while, the Main screen did not reacted to 'Refresh' and when you tried to watch something the Stream/VOD didn't loaded.

Etc:

  • Removed Card caching setting, because it's not necessary anymore.
  • Notification on the Main screen if you received a new app version.

If you have a question or feedback, head over to the Discord channel: https://discord.gg/zmNjK2S

v1.1 (2018.10.20)

General:

  • Exoplayer 2.9.0
  • targetSdkVersion 28
  • Replaced Picasso with Glide (~faster image loading)
  • Changelog implemented
  • Privacy Policy added
  • About: added Bottom bar icon descriptions

Major features:

  • VOD playback with Resume. Resume position is visible on the VOD Card.
  • User/Channel profiles: You can access Highlights, Past broadcasts and Uploads in the Channel Profile.
  • Follow and Favorite channel in the Channel profile
  • 'Past broadcasts' and 'Followed Channels' are accessible on the Main screen.
  • Search (Stream, Channel, Game). To have Voice Search, you have to grant RECORD_AUDIO permission.

UI:

  • Streamer uptime on Cards and on the Stream bottom bar.
  • Visual tweak in Stream category (blurred + dimmed background)
  • Auto refresh on app switch (Main screen and Game category)
  • Long press to Refresh on any Card, except Followed Channels and Settings
  • Long press on Followed Channels to favorite them. By favoriting a channel, makes it appear in front of the list, for easy access.
  • Thumbnails should show a much more recent image on Stream cards. If you turned Off Thumbnail caching (Settings->Miscellaneous) previously, you can turn it back ON.

Stream:

  • Improved stream startup time
  • Chat: send chat message. You can open the message window by the the icon, 'J' or 'F' keys. To close it: BACK, ESC or ENTER in an empty message window.
  • Channel profile icon added
  • 3 seconds delay on chat

Settings:

  • General settings: moved few settings into this sub category
  • General settings: Change Video card size (normal by default)
  • General settings: Hide any row on the Main screen
  • Stream settings: Video + Chat side by side (off by default)
